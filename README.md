Personal portfolio and blog of **Devesh Kumar** — Software Engineer II at Microsoft, B.Tech + M.Tech from IIT Bombay.

🌐 **Live site:** [devesh1102.github.io](https://devesh1102.github.io)

---

## About

A dark-themed personal website built with plain HTML, CSS, and JavaScript — no frameworks, no build step. Hosted on GitHub Pages.

---

## Sections

- **Hero** — Introduction, stats bar, and quick-action buttons
- **Experience** — Microsoft (MCI, Power Automate, Azure) + CME Group timeline
- **Skills** — Languages, Azure platform, frameworks, and engineering practices
- **Leadership & Impact** — Key achievements, mentoring, security certifications
- **Projects** — Personal and academic projects
- **Blog** — Engineering write-ups and career reflections
- **System Design Notes** — HLD deep dives (Kafka, Redis, Elasticsearch, Bloom Filters, etc.)
- **Contact** — Email form powered by Formspree

## Tech Stack

| Layer | Technology |
|---|---|
| Hosting | GitHub Pages |
| Markup | HTML5 |
| Styling | CSS3 (custom properties, grid, flexbox) |
| Scripting | Vanilla JavaScript |
| Fonts | Inter + JetBrains Mono (Google Fonts) |
| Icons | Font Awesome 6 |
| Markdown rendering | marked.js |
